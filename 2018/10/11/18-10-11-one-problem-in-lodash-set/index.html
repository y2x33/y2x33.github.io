lodash.set 方法的一个小坑

hellox
2018-10-11
javascript, lodash

问题描述

为何发生

如果 key 值中包含一个 number 时，默认这个 number 是一个数组的 index。于是就会尝试去处理一个数组。但是与此同时，我们传入的 number 超出了数组的最大长度，所以最终的返回值是一个空数组。

如何解决

1. 源对象已经包含这个字段

2. 使用 lodash.setWith ，并传入第四个参数
